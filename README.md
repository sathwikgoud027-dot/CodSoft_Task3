# 🍽️ Restaurant Menu UI

## 📖 About the Project

This project was developed as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to create an attractive and easy-to-use restaurant menu interface that enables users to browse dishes, view food details, customize their orders, and enjoy a seamless food ordering experience.

---

## 🖼️ Design Preview

![Restaurant Menu UI](CodSoft(Task03).png)

---

## 🎯 Design Objectives

- Easy food discovery
- Modern mobile interface
- Smooth ordering experience
- Attractive food presentation
- Intuitive navigation
- Clean visual hierarchy
- User-friendly customization
- Consistent design language

---

## ✨ Key Features

- Home screen with food categories
- Search functionality
- Popular dishes section
- Detailed product page
- Ingredients display
- Quantity selector
- Spice level customization
- Add to Cart option
